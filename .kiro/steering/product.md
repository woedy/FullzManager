# Product Overview

This is a full-stack web application for managing personal information and data records. The system appears to be designed for storing and organizing comprehensive personal profiles including:

- **Core Identity**: Names, aliases, demographics, and identification numbers
- **Contact Information**: Phone numbers, email addresses, and physical addresses  
- **Financial Data**: Credit scores and related information
- **Relationships**: Family members and associates
- **Assets**: Vehicle information and ownership records
- **Documentation**: File uploads and external links for evidence/verification

The application provides a dashboard interface for searching, filtering, and managing these records with full CRUD operations. It includes data import capabilities through management commands and supports bulk operations like clearing all records.

## Key Features
- Person profile management with comprehensive data fields
- Search and filtering across multiple data points
- File upload support for documentation
- Bulk data import from various file formats
- RESTful API with Django REST Framework
- Modern React frontend with responsive design
